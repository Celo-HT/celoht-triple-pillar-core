# CeloHT — Triple Pillar Ecosystem
Education • Agents • Reforestation

CeloHT is a community-driven initiative accelerating cUSD and Celo adoption in
Haiti using three foundational pillars:

1. **Education** — Blockchain literacy, financial inclusion training, certification.
2. **Agents** — A nationwide cUSD on/off-ramp network for real access.
3. **Reforestation** — Community-led ecological restoration funded transparently on-chain.

## Mission
Empower Haitians with digital finance, environmental recovery, and transparent blockchain tools.

## Vision
A financially included, digitally empowered, environmentally resilient Haiti.

## Core Values
- Transparency  
- Community ownership  
- Impact verification  
- Sustainability
